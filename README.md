# jscop
## Introduction

jscop is a static code analyser written in node js.

## Checks

* missing curling braces
* debugger statement
* duplicate property name
* empty block statements
* empty statements
* eval
* missing semicolon
* strict equality check
* trailing array comma
* with statement check
